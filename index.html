FoX: a Fortran XML library

FoX, (Fortran XML) is a library, written entirely in Fortran, designed to allow the easy use of XML from Fortran programs.

FoX caters for both XML output and input.

Output

  • A streaming output API which can generate all constructs permitted by XML and XML Namespaces (1.0 and 1.1). The output is guaranteed well-formed, and optional validity checks can also be made.

  • High-level output APIs to allow the generation of domain-specific XML documents, for:

    • CMLComp, (see also CML)

    • KML in beta, see WKML

Input

  • A fully validating parser, compliant with XML-1.0/1.1 and XML-Namespaces 1.0/1.1, xml:id, and xml:base

  • A SAX-like interface to this parser, offering all SAX-2 callbacks

  • A full W3C DOM interface, implementing all of DOM Core Level 1 and 2, and a substantial part of Level 3.

FoX is in use by several leading scientific simulation codes:

  • Abinit

  • CASTEP

  • Dalton

  • DL_POLY

  • GULP

  • OpenMopac

  • RMCProfile

  • SIESTA

  • VAMP

Background

Fortran and XML do not mix well. However, the vast majority of computational scientific codes in use, and under development, today, are written in Fortran, for a variety of reasons - and it would be nice if the output from these codes was available marked up in XML for better data management.

Equally, XML provides an increasingly common serialization format for exchange of data common to a domain. Fortran codes need to be able to produce and access information in XML formats.

There are a number of potential approaches to integrating Fortran codes into an XML-centric data world, including cross-language linking against existing XML libraries written in C; or having a separate post-processing stage which marks up unchanged text data from existing codes. However, with FoX, I have adopted the approach of writing the entire library in fully standard-compliant Fortran 95. This enables seamless use of FoX in any existing code, with no issues of cross-language compilation, or maintenance of separate processing stages.

The primary source code repository is available at http://www.uszla.me.uk/git/FoX.git, and may be browsed online.
